# HamR 管家应用 (app.hamr.store)

HamR 核心应用系统 - 五维家庭管理平台。

## 项目信息

- **项目编号**: PROJ-005
- **域名**: app.hamr.store
- **技术栈**: 
  - 前端: React + TypeScript + Vite
  - 后端: Rust + Axum + PostgreSQL + MongoDB

## 功能特性

- 五维管理体系（人时事物境）
- 家庭数据看板
- 大事记功能
- 数据导入/导出
- 端到端加密

## 项目结构

```
hamr-app/
├── frontend/           # React 前端
└── backend/            # Rust 后端
```
